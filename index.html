100% Free

No paid APIs ever. Uses OpenRouter free tier, Edge TTS, free Pexels & Pixabay footage, and GitHub Actions free tier.

AI Script Generation

OpenRouter AI (Llama 3.1, Mistral, Gemma — all free) writes professional hooks, narration & CTAs. Falls back to a smart template engine.

Viral Optimizer

Multi-Source Footage

Queries Pexels first, then falls back to Pixabay for portrait clips. Random clip start points keep content visually fresh every run.

Auto Thumbnails

Runs every 6 hours via GitHub Actions cron. Just add three secrets and enable the workflow — no server required.

python - <<'EOF'
from google_auth_oauthlib.flow import InstalledAppFlow
flow = InstalledAppFlow.from_client_secrets_file(
    "client_secret.json",
    scopes=["https://www.googleapis.com/auth/youtube.upload",
            "https://www.googleapis.com/auth/youtube.force-ssl"]
)
creds = flow.run_local_server(port=0)
import json
print(json.dumps({
    "access_token": creds.token,
    "refresh_token": creds.refresh_token,
    "token_uri": creds.token_uri,
}))
EOF
